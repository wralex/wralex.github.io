## Course Objectives

When you complete this course, you will be able to:

- Describe the purpose of Best Practice Customization Mode and Base
  Development Mode
- Create users, groups, and roles
- Create regular, join, and display-only forms
- Add, align, and organize fields
- Create control panels with navigation and application list fields
- Create active links, filters, and escalations
- Create packing lists and object definition files
- Analyze application performance
- Document applications
