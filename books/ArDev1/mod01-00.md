## Module Objectives

When you complete this module, you will be able to:

- Describe the functionality of Developer Studio
- Learn how to Authenticate and set Preferences
- Enable RSSO for Developer Studio
- Work with Developer Studio Perspectives
- Search for and view Object Relationships
- Use the Outline feature
- Switch between Best Customization Mode and Base Development Mode
- Convert Origin objects to Custom objects
- Describe Overlays and Granular Overlays
